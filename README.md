# Diabetes Risk Prediction

## Overview
This project predicts whether a person is likely to have diabetes using machine learning. It uses the Pima Indians Diabetes Dataset and a Logistic Regression model.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression model
- Model evaluation using accuracy score, confusion matrix, and classification report
- Predict diabetes risk for new patient data

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
- Git & GitHub

## Dataset
Pima Indians Diabetes Dataset
